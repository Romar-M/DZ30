# LMS API — Система управления обучением

Проект предоставляет API для онлайн-платформы обучения с возможностью размещения курсов и уроков, подписками, оплатой через Stripe и асинхронными задачами (Celery).

## Стек технологий
- **Django 5** + Django REST Framework
- **PostgreSQL** (основная БД)
- **Redis** (брокер Celery)
- **Celery + Celery Beat** (асинхронные и периодические задачи)
- **Stripe** (платежи)
- **JWT‑авторизация** (Simple JWT)
- **drf‑spectacular** (документация API)
